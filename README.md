## 🛠️ ポートフォリオ一覧

これまでに、アルゴリズムのスクラッチ実装や機械学習を用いたシステム開発などに注力してきた。以下に代表的なプロジェクトを紹介する。

### 1. [点群レジストレーション (pointcloud-registration-numpy)](https://github.com/koyama-mizuki/pointcloud-registration-numpy)
![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white) ![NumPy](https://img.shields.io/badge/NumPy-013243?style=flat&logo=numpy&logoColor=white)

NumPyのみを使用して、3D点群データの位置合わせ（ICPアルゴリズム等）をフルスクラッチで実装した。ライブラリに頼らず基礎的なアルゴリズムを数学・アルゴリズムレベルから理解して実装している。

<img src="https://github.com/koyama-mizuki/pointcloud-registration-numpy/blob/main/results/registration_result.png?raw=true" width="400" alt="レジストレーション結果">

### 2. [手話認識システム (JSL-Recognition)](https://github.com/koyama-mizuki/JSL-Recognition)
![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white) ![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?style=flat&logo=pytorch&logoColor=white) ![MediaPipe](https://img.shields.io/badge/MediaPipe-00A6B2?style=flat)

GNN（グラフニューラルネットワーク）とMLPを用いた日本の手話認識モデルである。MediaPipeを用いて骨格検出し、モデルの学習からリアルタイム推論までを一貫して実装した。

<img src="https://github.com/koyama-mizuki/JSL-Recognition/blob/main/assets/demonstration.gif?raw=true" width="400" alt="JSL動作デモ">
